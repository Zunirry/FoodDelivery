#Food Delivery

### Description
- FoodDelivery is an app to order your favorite food to your home. There is a different variety of restaurants and each of them have their own dishes.

#### Features
- App made with React Native.
- Geolocation.
- React Hooks.
- Google Api
- Integration with firebase.
- Stripe
- Dark mode
